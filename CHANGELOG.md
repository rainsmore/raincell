## 0.1.0

### New Features

- Created a readme including installation and a getting started tutorial ([#24](https://github.com/rainsmore/raincell/issues/24))

- Add contributing guide ([#26](https://github.com/rainsmore/raincell/issues/26))

- Add raw sample data for CML and Gauges over Douala ([#1](https://github.com/rainsmore/raincell/issues/1))

- Add function to assign gauges to links based on the distance to link path center ([#5](https://github.com/rainsmore/raincell/issues/5))

- Add wet dry classification algorithm based on TSLmax - RSLmin as signal variation ([#10](https://github.com/rainsmore/raincell/issues/10))

- Add basline computation algorithm based on last dry event ([#14](https://github.com/rainsmore/raincell/issues/14))

- Add constant WAA model ([#16](https://github.com/rainsmore/raincell/issues/16))

- Add k-R law based precipitation rate estimation using coeficients from Africa ([#20](https://github.com/rainsmore/raincell/issues/20))

- Create intermediate sample data for wetdry an baseline steps ([#21](https://github.com/rainsmore/raincell/issues/21))

- Create romulo heatmap plot for cml and gauges ([#28](https://github.com/rainsmore/raincell/issues/28))
